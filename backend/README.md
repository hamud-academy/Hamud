# Backend - E-Learning Platform

Backend logic runs via **Next.js API Routes** inside the `frontend/` project.

- **API Routes:** `frontend/src/app/api/`
- **Database:** PostgreSQL with Prisma ORM
- **Prisma Schema:** `frontend/prisma/schema.prisma` (to be added)

This folder is reserved for future shared backend utilities or if the API is extracted to a separate service later.
